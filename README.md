## 🚀 Overview

An interactive **Monthly Sales Dashboard** built with **Streamlit** that analyzes time-series sales data to uncover revenue trends, growth patterns, and store performance.

---

## ✨ Features

* 📂 Upload your own sales dataset
* 📊 Automatic revenue calculation
* 📈 Time-series analysis (monthly trends)
* 📉 Rolling average insights
* 📊 Cumulative revenue tracking
* 🏪 Multi-store comparison
* 🎛️ Interactive store filter

---

## 🛠️ Tech Stack

* Python 3.x
* Streamlit
* Pandas

* date
* store
* units_sold
* unit_price
